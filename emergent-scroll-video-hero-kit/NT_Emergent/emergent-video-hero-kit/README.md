# NomadaToast Emergent Scroll Video Hero Kit

This kit teaches people how to use Emergent to create a website where an uploaded MP4 becomes the main full-header hero, with the video progression controlled by scrolling.

## Included Files

- `NT Scroll Video Hero Prompt.txt`
  Upload this prompt into Emergent with your MP4.
- `Emergent Step-by-Step Guide.pdf`
  A visual walkthrough with screenshots, a clickable index, reference links, and NomadaToast socials.
- `README.md`
  These instructions.

## Simple Method

Use this when you already have:

- an MP4 video
- a website style you want Emergent to copy

Steps:

1. Open `https://app.emergent.sh/home`.
2. Select `Landing Page`.
3. Upload your MP4 video.
4. Upload `NT Scroll Video Hero Prompt.txt`.
5. Paste the website you want to copy into the chat. This can be a URL, copied website text, screenshots, or a direct reference.
6. Choose the settings you want.
7. Press enter.

The prompt tells Emergent to keep the reference site's colors, UI style, UX style, layout rhythm, and overall spirit. The only intentional change is the hero/header, which becomes a full video-led scroll experience.

## Enhanced Method

Use this when you want a more specific result.

Do the simple method, then add extra detail above or below the prompt:

- what the website is for
- the placeholder brand name you want
- the target audience
- the tone of the copy
- sections you want below the hero
- whether text should sit over the video or beside it
- timestamps or moments in the video that should line up with text changes

Example:

```txt
The website is for a premium productivity app for creative teams. Use the placeholder brand name "BrandName". Keep the same clean SaaS feel as the reference site. Use the attached MP4 as the main full-screen scroll-controlled hero.
```

## Result Checks

After Emergent generates the page, check:

- the MP4 is the main hero visual
- the video is not a small side card or preview tile
- scroll controls the video progression, or the fallback still feels scroll-linked
- the copied site's colors and UI style are still recognizable
- the real brand and product names have been replaced with placeholders
- the first viewport feels like the original site rebuilt around the video

If the style is too different, send:

```txt
Keep the same site style more strictly. Match the reference colors, typography, spacing, UI components, and UX rhythm. Only change the hero/header so the attached video becomes the dominant scroll-controlled hero.
```

If the video is too small, send:

```txt
Make the attached video dominate the entire hero/header area. Do not place it as a card, side image, preview tile, or secondary section.
```
