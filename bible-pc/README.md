# Bible PC

This is a Windows desktop wrapper for the hosted Bible website at `https://junha2010.github.io/bible/`.

The app opens the Bible reader in its own desktop window. Links outside the Bible website open in the user's normal browser.

## Build EXE From PowerShell

```powershell
.\build-exe.ps1
```

The portable Windows executable is written to:

```text
dist\BibleViewer-Windows-1.0.0.exe
```
